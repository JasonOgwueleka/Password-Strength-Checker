# Password Strength Checker

A simple Python program that collects and validates user login information.  
It checks for spaces, enforces password rules, and stores valid login details in a dictionary.

---

## Features:
- Prompts the user for **username**, **password**, and **email**
- Ensures no spaces are used in any input
- Validates that passwords:
  - Are at least **8 characters long**
  - Contain **uppercase**, **lowercase**, and **numeric** characters
- Provides clear feedback messages
- Stores valid user data safely in a **dictionary**
- Demonstrates **Python logic and validation techniques**

---

## Concepts Used:
- Conditional statements (`if`, `else`)
- String handling and validation
- Dictionaries for structured data
- Basic input/output
- Password strength checking logic

---

## How to Run:
1. Clone this repository:
   git clone https://github.com/JasonOgwuleka/Password-Strength-Checker.git
2. Navigate into the project folder:
   cd Password-Strength-Checker
3. Ensure you have Python installed.
4. Run the script:
   python passwordStrength.py
